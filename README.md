# Asteroids+ Capstone Project

This repository contains the source code for the Asteroids+ Capstone project made by Roberto Manra and Alex Zhou. This project was made with the guidance of Lidia Morrison in the Spring of 2026.

Here you will find the source code for this project. It is possible to get the source code running on any machine but will require some prior setup. We recommend following online guides to aid the installation process of Unity, Unity Hub, and any code editor of your choice.

If you simply want to see the end product, head over to our itch.io page (https://rmanra22.itch.io/capstone-cpsc-491-spring-26) where you can download the zip file, extract it, and play the game.


## Requirements

Unity 6

Unity Hub

Visual Studio or any C# compatible code editor

## Controls

Up Arrow — Accelerate

Left Arrow — Turn Left

Right Arrow — Turn Right

Space — Shoot
